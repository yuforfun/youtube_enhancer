語言清單 A：原文顯示語言

設定您能看懂的語言。當影片有這些語言字幕時，系統將直接顯示依列表順序檢查並觸發匹配的語言。

💡 提示： 當字幕是您設定的語言時，系統會直接顯示原文，不會消耗 API 配額。

語言清單 B：自動翻譯與 Prompt 管理

設定需要自動翻譯的語言，並為每個語言自訂 Prompt。系統會依列表順序檢查並觸發第一個匹配的語言。

⚠️ 優先級說明： 系統會依序檢查列表，並觸發第一個匹配的翻譯。拖曳項目可調整優先序。
